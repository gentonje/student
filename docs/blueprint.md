# **App Name**: CounsellorAI

## Core Features:

- Dynamic Forms: Interactive, branching question format, mimicking Typeform's style.
- Adaptive Questioning: AI analyzes user inputs in real-time and personalizes subsequent questions for HIV counseling.
- AI-Powered Advice: AI tool provides personalized advice and risk assessments based on user's responses during the counseling session.
- Progress Tracking: Progress indicator to show how far along the user is in the counseling session.

## Style Guidelines:

- Primary color: A calming and supportive blue (#4A8FE7) to create a trustworthy atmosphere.
- Background color: Light, desaturated blue (#E6F0FA) for a clean and unobtrusive backdrop.
- Accent color: A gentle green (#7ACC7A) to highlight positive affirmations and key information.
- Use a clean, sans-serif font for readability and a modern feel.
- Simple, clear icons to represent different sections of the counseling process.
- Focus on single-question presentation with clear, prominent answer options.
